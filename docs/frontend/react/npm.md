# React + npm Quick Notes

- Initialize a package file: `npm init`
- Install dependencies: `npm install`
- Add scripts inside `package.json`, e.g.

```json
{
  "scripts": {
    "start": "npm run dev",
    "build": "npm run build"
  }
}
```

- Install Parcel for quick prototyping: `npm i parcel`
- Run Parcel entry point: `npm run start` with `"start": "parcel app.js"`
